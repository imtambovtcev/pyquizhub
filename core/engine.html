class pyquizhub.core.engine.engine.QuizEngine(quiz_data)[source]

Core quiz engine that manages quiz sessions and flow.

This class handles: - Quiz data validation and loading - Session management for multiple users - Question progression logic - Score calculation - Answer processing

  • sessions (dict) – Active quiz sessions by session ID
